Palmtree Application Common Class Library
====

.NETでよく使われるアプリケーションの共通機能のライブラリです。

## Description
.NETランタイムで実装されていない、よく使われるアプリケーションの共通機能を提供するライブラリです。以下のようなクラスが提供されています。
- WEBサイトのドキュメントについてリンクをたどりつつ一定の情報を収集するアプリケーションの基本クラス。

## Requirement
- .Net Framework 4.6
